---
description: Sync changes to GitHub
---
Whenever a task is completed or a significant milestone is reached, follow these steps to ensure the GitHub repository is up to date.

// turbo
1. Stage all changes:
   `git add .`

2. Commit the changes with a descriptive message:
   `git commit -m "Descriptive message about the changes"`

// turbo
3. Push the changes to the remote repository:
   `git push origin main`
